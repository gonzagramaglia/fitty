# Project Overview

## About the Project

Fitty is an AI-powered cat health tracker that estimates Body Condition Score (BCS) from two photos (top and side view). It classifies cats as underweight, healthy, or overweight, explains the visual reasoning, and provides simple care recommendations. Fitty also tracks results over time to help owners monitor weight changes and spot health issues early.

## The Problem It Solves

Monitoring a cat's weight and health can be challenging for pet owners. It's often hard to tell if a cat is becoming overweight or losing too much weight just by looking at them daily, and traditional scales aren't always cooperative. Fitty eliminates the guesswork by using computer vision to objectively score the cat's Body Condition Score based on veterinary standards. Owners get instant feedback and actionable recommendations without needing to stress the cat with a vet visit just for a weight check.
